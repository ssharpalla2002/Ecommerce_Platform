# E-commerce Platform

## Overview

This **E-commerce UI** is built from scratch which includes essential UI components such as a navbar, product slider, category pages, single product pages, newsletter, login/register forms, and shopping cart. The goal is to practice React component structuring, modular styling, and responsive design while creating a real-world eCommerce layout.

## Technical Working

The project is built with **React.js** and **Styled Components** for modular, reusable styling.Key features include:

- **Navbar:** Responsive header with logo, navigation links, and cart icon.  
- **Product Slider:** React-based slider for featured products with smooth transitions.  
- **Categories & Product Lists:** Dynamic category pages and product cards with hover animations.  
- **Single Product Page:** Displays product images, description, price, and add-to-cart button.  
- **Newsletter & Footer:** Styled newsletter sign up and footer with links and social icons.  
- **Login & Register Forms:** Clean React forms for authentication UI.  
- **Shopping Cart:** Interactive cart showing selected products and totals.  
- **Responsive Layouts:** Fully responsive design using media queries and flexible grids/flex layouts.
